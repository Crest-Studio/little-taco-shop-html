# Little Taco Shop

A simple multi-page restaurant website built using pure HTML.

## Description
This project is a practice HTML website created while learning HTML fundamentals.  
It includes multiple pages like Home, Store Hours, and Contact Us.

## Technologies Used
- HTML5

## What I Learned
- Semantic HTML elements
- Linking multiple pages
- Forms and tables
- Accessibility basics (alt, labels, semantic tags)

## Project Status
Completed as part of HTML learning practice.
